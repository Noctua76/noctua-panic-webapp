# noctua-panic-webapp
Noctua Panic Web App (V1) Mobile-first σύστημα άμεσης ειδοποίησης για εταιρείες security. Με ένα πάτημα ενεργοποιεί 3 κλήσεις και 3 SMS, εμφανίζει live status και καταγράφει το περιστατικό. Περιλαμβάνει AI incident reporting και δυνατότητα κλιμάκωσης σε V2 mobile app και V3 on-screen AI.
